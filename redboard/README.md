Redboard Folder Overview
==========================

This folder contains the Code Composer Studio project for programming the Roambot's TM4C Redboard microcontroller.

Contents:

1. **roambot**: This is the Code Composer Studio project folder. It includes all the necessary source code and project settings for compiling and programming the TM4C Redboard microcontroller.

The Redboard microcontroller is responsible for low-level operations such as motor control, serial communication with the Raspberry Pi, and interfacing with other hardware. This folder is essential for compiling and deploying firmware to the microcontroller.
